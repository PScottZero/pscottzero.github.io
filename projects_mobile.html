Apogee Artist


The spiritual successor to my previous program, Java Paint. I decided to take my ideas from Java Paint and make a new and heavily improved program. The program is entirely rewritten and now uses Kotlin. Also, the user interface is now completely resizable unlike in Java Paint. With a wide variety of advanced tools and features, I feel that this is the closest I have come to making a passable piece of illustration software. I'm not saying that this program is going to compete with Adobe or anything, but at least the program is exciting to use!

[Programmed in Koltin]


Bitwise Game Maker


I originally created this project for the Spring of 2018 Hack PSU. I decided after the Hackathon to refine its design and make it work better. Bitwise Game Maker allows you to customize the blocks in the level and your character. Each block can either be a, and background block (does not interact with the player), platform block (the player can stand on it), or enemy block (makes the player lose a life). The main "goal" of the game is to get to the other side of the level. However, since you are in control of the level design, you can make this goal harder or easier to achieve.

[Programmed in Java]

ExGraph


Watching my trusty TI-84 drawing beautiful graphs always made me think "could I make a program that also does this?" The result is a program that takes in Reverse Polish Notation equations and spits out either cartesian, polar, or parametric graphs.

Lucas-Lehmer Primality Test


Want to see how well your computer can handle calculated Mersenne Primes?!? (Large Prime in form 2ⁿ-1) Then look no further. Using the Lucas-Lehmer primality test, this program is able to calculate exceedingly large prime numbers.

STP Ground


During my fall semester of 2017, I was part of the Student Space Program Laboratory's student training program (STP). There were three subgroups in this program. Two of which built small rockets that would contain a payload. This payload would send data to the last group, the ground station. I found myself in that group, and ended up designing the GUI used to display the information received from the payload. Due to communication errors on the day of launch, I was never actually able to see this program run for real. However, it should theoretically work.

[Programmed in Python]
